# 💸 Expense Tracker App with Data Science & Machine Learning

## 🚀 Overview

The **Expense Tracker App** is a full-stack data-driven application that helps users track, analyze, and predict their expenses. It combines **data analysis, visualization, machine learning, and web development** into a single interactive platform.

This project is designed to demonstrate real-world skills required for **Data Analyst, Business Analyst, and Machine Learning roles**.

---

## 🎯 Problem Statement

Managing personal finances is difficult without proper tracking and insights. Most individuals:

* Do not track spending patterns
* Overspend without realizing
* Lack forecasting for future expenses

---

## 💡 Solution

This application provides:

* Expense tracking system
* Interactive dashboard
* Budget monitoring
* Overspending detection
* Machine learning-based forecasting
* PDF report generation

---

## ✨ Key Features

### 🔐 Authentication System

* User registration and login
* Session management
* Secure data handling

### 💾 Database Integration

* SQLite database for persistent storage
* User-specific expense tracking

### 📊 Data Analysis & Visualization

* Category-wise spending analysis
* Monthly expense trends
* Payment method distribution
* KPI metrics (Total, Average, Max)

### ⚠️ Smart Insights

* Overspending detection
* Budget analysis
* Advanced recommendations

### 🔮 Machine Learning

* Linear Regression for future expense prediction
* ARIMA model for time-series forecasting

### 📄 Report Generation

* Downloadable PDF reports

### 🌐 Deployment

* Fully deployed using Streamlit Cloud

---

## 🛠️ Tech Stack

* **Frontend:** Streamlit
* **Backend:** Python
* **Database:** SQLite
* **Data Analysis:** Pandas, NumPy
* **Visualization:** Matplotlib, Seaborn
* **Machine Learning:** Scikit-learn, Statsmodels (ARIMA)
* **Reporting:** ReportLab

## ⚙️ Installation & Setup

### 1. Clone Repository

```
git clone https://github.com/varda24/expense-tracker-ml.git
cd expense-tracker-ml
```

### 2. Create Virtual Environment

```
python -m venv venv
venv\Scripts\activate   # Windows
```

### 3. Install Dependencies

```
pip install -r requirements.txt
```

### 4. Run Application

```
streamlit run app.py
```
